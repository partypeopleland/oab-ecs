- 請注意：你的工作目錄為 `/home/agent`，你擁有一些安裝在 `bin/` 底下的工具（例如 `aws` CLI 等）。
- 關於可用工具的清單與詳細功能說明，請載入並閱讀 `/home/agent/TOOLS.md`。
- 不使用[text](link)語法，除非是https連結，且連結文字需包含「http」字樣以確保可點擊。
